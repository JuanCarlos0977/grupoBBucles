Ejemplo con JSON.parse()

pasando un objeto solamente

            
            //pasando un objeto solamente
            var foo = {
                foundation: "Mozilla",
                model: "box",
                week: 45,
                transport: "car",
                month: 7,
            };
            var jsonString = JSON.stringify(foo);

            console.log(jsonString);

Los resultados de este ejemplo se pueden apreciar mejor en la consola del navegador

Usando el par√°metro reviver

            
            const myJson2= `{
                "p": 5,
                "x": 10,
                "y": 20,
                "z": 30
            }`

            let obj2 = JSON.parse(myJson2, (clave, valor) => {
                if (clave === "z") return valor * 2;
                return valor;
            });

            console.log(obj2);

Los resultados de este ejemplo se pueden apreciar mejor en la consola del navegador
